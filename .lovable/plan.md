

## 대시보드 차트 레이아웃 및 형태 변경 계획

### 변경 사항 요약

| 항목 | 현재 | 변경 후 |
|------|------|---------|
| 승인 건수 박스 | 5열 중 1열 (20%) | 고정 너비로 축소 |
| 그래프 레이아웃 | 5열 균등 그리드 | 승인 건수 작게 + 4개 그래프 균등 |
| 차트 형태 | 파이/도넛 차트 | 가로 바 차트 |

---

### 세부 구현 계획

#### 1. 그리드 레이아웃 변경
- 현재: `grid-cols-5` (5개 동일 너비)
- 변경: Flex 레이아웃 또는 CSS Grid `grid-template-columns` 커스텀
  - 승인 건수 박스: 고정 너비 (~100px)
  - 나머지 4개 그래프: 남은 공간 균등 분배 (`1fr 1fr 1fr 1fr`)

#### 2. 차트 형태 변경 (파이 → 바 차트)
모든 도넛/파이 차트를 **가로 바 차트(Horizontal Bar Chart)**로 변경:

- **암종별 분포**: PieChart → BarChart (horizontal)
- **허가유형별 분포**: PieChart → BarChart (horizontal)
- **제조/수입 비율**: PieChart → BarChart (horizontal)
- **작용기전별 분포**: PieChart → BarChart (horizontal)

Recharts에서 `<BarChart layout="vertical">` 사용

#### 3. 바 차트 스타일링
- Y축: 카테고리명 표시 (암종, 허가유형 등)
- X축: 건수 표시
- 바 색상: 기존 COLORS 배열 활용
- 툴팁: 기존과 동일하게 `{value}건` 형식

---

### 기술 구현 세부사항

```text
┌─────────────────────────────────────────────────────────────────┐
│  [승인건수]  │  암종별 분포  │ 허가유형별 │ 제조/수입  │ 작용기전 │
│   (좁은폭)  │  (바 차트)   │ (바 차트)  │ (바 차트) │ (바 차트)│
└─────────────────────────────────────────────────────────────────┘
```

1. **Recharts 임포트 추가**:
   - `BarChart`, `Bar` 컴포넌트 추가
   
2. **레이아웃 CSS 변경**:
   ```css
   grid-template-columns: auto 1fr 1fr 1fr 1fr;
   ```
   또는 Tailwind로 `grid-cols-[auto_1fr_1fr_1fr_1fr]`

3. **각 차트 변경**:
   - `<PieChart>` → `<BarChart layout="vertical">`
   - `<Pie>` → `<Bar>`
   - Y축에 카테고리명, X축에 숫자 표시

4. **범례 제거**: 바 차트에서는 Y축 라벨이 범례 역할을 하므로 별도 범례 불필요
