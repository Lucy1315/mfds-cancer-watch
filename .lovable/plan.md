

## 엑셀 요약 시트 스타일 변경 계획

### 변경 요청 사항
1. **"제품별 상세 목록" 영역의 행 높이 증가** - 가시성 향상
2. **색상 변경: 블루 → 그레이 계열** - 전체 스타일 톤 변경

---

### 현재 상태 분석

| 항목 | 현재 값 | 위치 |
|------|---------|------|
| 요약 시트 행 높이 | 제목 26pt, 나머지 **20pt** 균일 | 96행 |
| 헤더 배경색 | `#2563EB` (파란색) | STYLES.header |
| 섹션 헤더 배경색 | `#DBEAFE` (연한 파란색) | STYLES.sectionHeader |
| 서브 헤더 배경색 | `#3B82F6` (파란색) | STYLES.subHeader |
| 테두리 색상 | `#1E40AF`, `#3B82F6` 등 파란 계열 | 각 스타일 border |

---

### 세부 구현 계획

#### 1. 행 높이 변경 (요약 시트)
현재 모든 행이 20pt로 균일하게 설정되어 있음. "제품별 상세 목록" 헤더 이후의 데이터 행 높이를 증가시킴.

**변경 로직:**
- 제목 행: 26pt (유지)
- 일반 행: 20pt (유지)
- 제품 목록 헤더 행: 28pt (증가)
- 제품 목록 데이터 행: 26pt (증가, 현재 20pt → 26pt)

#### 2. 색상 변경 (블루 → 그레이)
모든 스타일의 파란색 계열을 그레이 계열로 변경:

| 스타일 | 현재 색상 | 변경 후 색상 |
|--------|----------|-------------|
| **header** 배경 | `#2563EB` (블루) | `#374151` (다크 그레이) |
| **header** 테두리 | `#1E40AF` (다크 블루) | `#1F2937` (더 진한 그레이) |
| **sectionHeader** 배경 | `#DBEAFE` (연한 블루) | `#E5E7EB` (연한 그레이) |
| **sectionHeader** 폰트 | `#1E40AF` (다크 블루) | `#374151` (다크 그레이) |
| **sectionHeader** 테두리 | `#3B82F6` (블루) | `#9CA3AF` (중간 그레이) |
| **subHeader** 배경 | `#3B82F6` (블루) | `#4B5563` (그레이) |
| **subHeader** 테두리 | `#1E40AF` (다크 블루) | `#374151` (다크 그레이) |

---

### 수정 파일

| 파일 | 변경 내용 |
|------|----------|
| `src/utils/excelExport.ts` | STYLES 상수의 색상 값 변경, 요약 시트 행 높이 로직 수정 |

### 기술 구현 세부사항

1. **STYLES 상수 수정**: 모든 파란색 RGB 값을 그레이 계열로 교체
2. **행 높이 로직 변경**: `summarySheet['!rows']` 설정 시 "제품별 상세 목록" 헤더 행 인덱스를 찾아 해당 행부터 높이를 26pt로 설정
